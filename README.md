# 🍎 Gran Rifa Apple Bot

Bot de Telegram para gestionar la rifa de productos Apple.

## Variables de entorno en Railway

| Variable | Valor |
|----------|-------|
| `BOT_TOKEN` | Token de @BotFather |
| `ADMIN_ID` | Tu ID de Telegram (5012720317) |

## Funciones del bot

### Para participantes:
- Ver boletos disponibles / reservados / vendidos
- Reservar un boleto eligiendo número
- Elegir forma de pago (contado o cuotas)
- Ingresar nombre y celular
- Recibir confirmación automática

### Para el admin (tú):
- Recibir notificación instantánea de cada reserva
- Confirmar pago con un botón ✅
- Liberar boleto si no paga ❌
- Ver panel completo con /admin
- Liberar cualquier número manualmente
